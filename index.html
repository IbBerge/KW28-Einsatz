☁️ Online-Synchronisation mit GitHub

✓ Verbunden mit GitHub
—
📖 Einrichtung – so geht's (einmaliger Aufwand, ca. 5 Minuten)

Was du brauchst: Ein GitHub-Konto (hast du), ein privates Repository und einen sogenannten „Personal Access Token" (PAT). Der Token erlaubt diesem Tool, deine Daten dort zu speichern und zu laden – wie ein Hausschlüssel, der nur in deinen Browser kommt.

Schritt 1: Privates Repository anlegen

  1. Gehe auf github.com/new
  2. Name z.B. arbeitseinsatz-daten
  3. Wichtig: „Private" auswählen! (damit deine Daten nicht öffentlich sind)
  4. Häkchen bei „Add a README file" setzen (damit das Repo nicht leer ist)
  5. „Create repository" klicken

Schritt 2: Personal Access Token erstellen

  1. Gehe auf github.com/settings/personal-access-tokens/new
  2. Token-Name: z.B. Arbeitseinsatz Planer Sync
  3. Ablauf (Expiration): z.B. 1 Jahr (danach musst du den Token erneuern)
  4. Bei „Repository access": „Only select repositories" → dein neues Repo auswählen
  5. Bei „Repository permissions" → „Contents" → „Read and write" auswählen
  6. Ganz unten „Generate token" klicken
  7. Der Token wird genau einmal angezeigt – kopiere ihn sofort! Er fängt mit github_pat_… an

Schritt 3: Hier eintragen

Trage Benutzername, Repository-Name und den Token unten ein und klicke „Verbinden". Fertig.

🔒 Sicherheit: Der Token wird nur lokal in deinem Browser gespeichert (localStorage). Er verlässt dein Gerät nur, um direkt mit GitHub zu kommunizieren. Niemals weitergeben! Wenn du den Token verlierst oder das Gefühl hast, er ist kompromittiert: in den GitHub-Einstellungen widerrufen und einen neuen erstellen.

Einsatztag abschließen

Wähle pro Projekt, was als Nächstes passiert. Notizen werden mit Datum als zusätzliche Zeile zur Projekt-Notiz hinzugefügt.

Namen aus Bild übernehmen

  1. claude.ai öffnen – mit Klick auf den Button unten. Der passende Prompt wird automatisch in die Zwischenablage kopiert.
  2. Dort das Bild der Anmeldung hochladen (Büroklammer-Symbol oder per Drag & Drop).
  3. Den Prompt einfügen (Strg+V) und absenden.
  4. Claude gibt eine Komma-Liste der Namen zurück. Diese kopieren.
  5. Zurück hierher und in das untere Feld einfügen.
